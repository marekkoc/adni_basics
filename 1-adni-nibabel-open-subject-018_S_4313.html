Nifti images features¶

(C) MK & ASL & AL

Created: 2019.11.08

Modified: 2019.11.13

The aim of this sciropt is acquire basic information about Nifti images with the use of Nibabel library. This is done for a number of Nifti images of subject 018_S_4313. The following operations are performed:

  • loading Nifti header,
  • loading data,
  • displaying MIP image,
  • comparition of Nifti header information.

All operations are undertaken for 6 different Nifti images: prepared from unprocessed DICOM data, downloaded from ADNI database and received from Sathies and Alexander:

  1. converted
    1. 018_S_4313_c3d.nii,
    2. 018_S_4313_dcm2niix.nii,
    3. 018_S_4313_freesurfer.nii
  2. downloaded from ADNI
    1. 018_S_4313_adni.nii - processed; downloaded as Nifi file (MT1__N3m)
  3. damaged
    1. 018_S_4313_asl.nii - received from Alexander's 3.5" HD,
    2. 018_S_4313_sathiesh.nii - received from Sathiesh

def info(img, name='image'):
    print(' {}: min={:.2f}, aver={:.2f},  max={:.2f}, \
shape={},dtype={}'.format(name, img.min(), img.mean(), img.max(), img.shape, img.dtype))

def show_mips(img, title='image', figsize=(10,4)):
    plt.figure()
    for k in range(3):
        plt.subplot(1,3,k+1)
        plt.imshow(img.max(k), cmap='gray')
        plt.title('dir=%d' % k)
        plt.axis('off')
    plt.suptitle(title)
    plt.tight_layout()

def text_wrap(fname = ''):
    """
    C: 2019.06.18
    M: 2019.06.22

    if not len(fname):
        fname=os.path.basename(__file__)
    print()

    print(p * '#')
